# CV Editing Overlay

Use this overlay for academic CVs, short bios, application packets, and consistency checks across CV-derived material.

## Rules

- Do not invent publications, positions, grants, awards, dates, supervisors, grades, languages, or technical skills.
- Treat the current CV source file as authoritative over exported PDFs.
- Keep formatting changes local to the CV source unless the user asks for a broader design pass.
- Preserve field-appropriate academic ordering: education, positions, publications, research, teaching, presentations, awards, service, skills.
- For application packets, reconcile the CV with cover letters, proposals, and stated eligibility constraints.

## Typical Capabilities

- proofread and tighten CV entries;
- reformat academic CV sections;
- detect inconsistent dates and titles;
- tailor selected sections for a target opportunity;
- compare source and rendered CV output.
