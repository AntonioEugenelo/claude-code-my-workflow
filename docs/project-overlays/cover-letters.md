# Cover Letters Overlay

Use this overlay for cover letters, statements of purpose, fellowship applications, exchange applications, and short academic application documents.

## Rules

- Preserve the user's authentic academic circumstances and avoid inventing qualifications, affiliations, offers, awards, or timelines.
- Treat application prompts, calls, program pages, and uploaded guidance as binding source material.
- Keep each document targeted to the institution and opportunity; do not reuse generic paragraphs without checking fit.
- For final documents, check alignment across cover letter, CV, proposal, references, and stated dates.
- If the output will be submitted externally, run an explicit review pass before claiming it is ready.

## Typical Capabilities

- draft or revise cover letters;
- tailor statements to a call or program;
- check consistency against a CV or proposal;
- compress text to page or word limits;
- produce application-specific review notes.
