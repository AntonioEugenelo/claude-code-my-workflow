# Teaching Slides Overlay

Use this overlay for courses, lecture notes, problem sets, teaching slides, and classroom examples.

## Rules

- Treat source `.tex`, `.qmd`, `.R`, `.py`, `.jl`, `.m`, and figure scripts as authoritative.
- Keep notation consistent across lectures, problem sets, solutions, and slides.
- Prefer pedagogical clarity over paper-style compression.
- Check that examples, equations, and problem-set answers are internally consistent before rendering final outputs.
- When modifying slides, inspect the rendered output if possible; slide source that compiles is not automatically visually acceptable.

## Typical Capabilities

- create or revise lecture slides;
- convert Beamer to Quarto;
- draft problem sets and solutions;
- verify equations and notation;
- review slide pacing and visual clarity.
