Codex-first migration. This branch keeps the academic workflow structure, scripts, templates, and review discipline from the original repository, but rewrites the control plane around Codex instead of Claude-specific hooks and slash commands.

A starter kit for academic work with LaTeX, Beamer, Quarto, R, Python, and MATLAB under a Codex-native workflow. The repo now uses AGENTS.md plus explicit workflow docs for planning, verification, review routing, capability mapping, and durable project state.

What changed in this branch

  • Codex entrypoint the primary instruction file is now AGENTS.md, not CLAUDE.md
  • Explicit workflows planning, orchestration, review routing, session logging, and capabilities are documented in docs/codex-workflows/
  • Plain-language task model instead of slash commands, users ask directly to compile, review, proofread, translate, validate citations, or run analysis
  • Verification-first loop Codex is expected to implement, verify, review, fix, and re-check rather than rely on runtime hook enforcement
  • Preserved automation existing scripts, templates, plans, session logs, and repository structure remain intact
  • Legacy reference retained the old .claude/ tree still exists for comparison, but it is no longer authoritative on this branch
